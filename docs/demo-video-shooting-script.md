# 撮影台本（CursorFly 用・操作する人向け）

字幕・シーン構成は [demo-video-script.md](demo-video-script.md) 参照。この文書は**録画しながら画面を操作する人**の手順書。
テイクごとに録画を分ける（ミスったらそのテイクだけ撮り直し）。全テイク合わせて素材 10〜15 分 → 編集で 2:30 に切る。

## 事前準備（録画前に1回だけ）

- [ ] Discord Web で `minutes-agent-demo` の **#一般** を開き、一番下までスクロール
- [ ] OS とブラウザの通知をオフ（撮影中のポップアップ防止）
- [ ] ブラウザウィンドウはフルスクリーンでなくてよい（1280〜1500px 幅くらいが文字が読みやすい）
- [ ] MIYA_IF さんとボイスチャンネルに入っておく（Take 3 用）
- [ ] CursorFly: マイク **OFF**（動画の音声は編集で付けるので不要）、システム音声 OFF

## 撮影のコツ（全テイク共通）

- CursorFly のズームは**カーソルとクリックに追従**する。見せたい行の上にカーソルを置いて **2秒静止**が基本動作
- コマンド応答が返った直後は**3秒待ってから**次の操作へ（編集で切りやすくなる）
- 操作を急がない。ゆっくりで OK（速度は編集で調整できる）

---

## Take 1: /setup — エージェントがリポジトリを読んで用語を学習（目安 1.5分）

1. 録画開始
2. メッセージ欄をクリック → `/setup` と入力 → 候補から **/setup** をクリック
3. repo 欄に `hinapupil/minutes-agent` と入力 → **Enter**
4. 「考え中…」の間はカーソルを応答の位置に置いて待つ（20〜40秒。長くても編集で切るので回し続ける）
5. 「✅ 用語集を学習しました（60語 …）」が出たら:
   - **「MiyaIF (みやいふ)」の行にカーソルを置いて2秒静止**（ズームさせる）
   - リストを上から下へゆっくりカーソルでなぞる
6. 3秒静止 → 録画停止

## Take 2: /ask — 過去の議事録に質問（目安 1分）

1. 録画開始
2. `/ask` → 候補クリック → question 欄に `次回のリリースはいつ？` → **Enter**
3. 「考え中…」を待つ（数秒〜30秒）
4. 「次回のリリースは金曜日に予定されています。」が出たら**回答の上で2秒静止**
5. 3秒静止 → 録画停止

## Take 3: 録音フロー — /join → 会議 → /stop（目安 2分）★音声読み合わせ

1. 録画開始
2. `/join` → **Enter**（ボイスチャンネルに入った状態で）
3. 「録音しますか？」の確認が出たら **[録音する] ボタンをクリック**（ボタンにカーソルを置いてからクリック）
4. 「⏺️ 録音を開始しました」の公開通知の上で2秒静止
5. **読み合わせ（マイクに向かって、普通の会話のテンポで）**:
   - hina: 「じゃあ定例始めます。**次回リリースは金曜にします**」
   - MIYA_IF: 「了解です。**プロトペディアの公開は hina さん担当**でお願いします」
   - hina: 「はい。**みやいふさんは動画編集をお願いします**」
   - MIYA_IF: 「わかりました。今日はここまでにしましょう」
6. `/stop` → **Enter** → 「議事録を作成中です…」通知の上で2秒静止
7. 3秒静止 → 録画停止
8. **録画を止めたまま議事録の投稿を待つ**（2〜5分。#議事録 に投稿される）

## Take 4: 議事録のお披露目（目安 1.5分）★一番の見せ場

1. **#議事録** チャンネルを開いてから録画開始
2. いま投稿された議事録の先頭までスクロール
3. 上から**ゆっくり**スクロールしながら、カーソルで順になぞって各2秒静止:
   - 概要
   - **話者名が表示名（hina / MIYA_IF）になっている箇所**
   - **「プロトペディア」「みやいふ」が正しく表記されている箇所**（用語補正の証拠）
   - 決定事項（「次回リリースは金曜」）
   - アクションアイテム（担当者つき）
   - **継続確認**（前回の会議との突き合わせ）← ここが差別化ポイント、長めに3秒
4. 録画停止

## Take 5: /actions と完了報告（目安 1分）

1. #一般 に戻って録画開始
2. `/actions` → **Enter** → 一覧が出たら Take 3 の会議で生まれたアイテム（「動画を編集する / みやいふ」等）の上で2秒静止
3. その ID をコピーして `/action-done` → id 欄に貼り付け → **Enter**
4. 「✅ … を完了にしました」（公開メッセージ）の上で2秒静止
5. 録画停止

## Take 6: ランディングページ（目安 1分）

1. 新しいタブで <https://minutes-agent-interactions-uj5jsuakcq-an.a.run.app/> を開いてから録画開始
2. 上から順にゆっくりスクロール、各所で2秒静止:
   - 「✅ このサービスは稼働中です」バッジ
   - 動作確認の手順
   - なぜ「デモサーバー招待」方式か
   - 構成（すべて Google Cloud）
   - リンク（設計文書 design.md の上にカーソル）
3. 録画停止
